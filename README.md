**镜渊** 是一个基于大语言模型的多智能体平行人生模拟系统。  
**织渊** 是其联网扩展形态，将千万条个人世界线编织为一张因果纠缠的命运之网。

---

## 🌌 项目愿景

我们相信，每个人心中都住着一个未被选择的自己。  
**镜渊·织渊** 并非一款游戏，而是一台 **存在主义共鸣机器**。

- 在 **镜渊（单机）** 中，你创建另一个自己的「镜中人」，观看他在真实历史与虚拟未来中，依据 AI 驱动的人格、记忆与情感，做出属于他自己的选择。你有且仅有 **三次干预机会**——每一次干预，都将引发不可预知的代价涟漪。
- 在 **织渊（联网）** 中，无数用户的镜中人在历史节点上擦肩而过，产生 **因果纠缠**。你们的个人世界线将悄然交织，共同塑造出共享的 **共鸣域**，甚至涌现出只可阅读的 **神话域** 传奇。

我们试图回答三个问题：
1. 如果当初……会怎样？
2. 我的选择真的属于我吗？
3. 我与他人如何在看不见的地方相互塑造？

---

## 🧬 核心特性

### 🪞 镜渊（单机模拟引擎）
- **五维人格模型**（OCEAN）随人生经历动态演化。
- **三层记忆系统**（感知/工作/长时记忆），模拟真实的遗忘与闪回。
- **PAD 三维情感模拟**，情绪影响决策权重。
- **有限理性决策引擎**——镜中人不会寻找最优解，只会做出「足够好」的选择。
- **三次干预权**：念头植入、环境暗示、梦境传递。每次干预都伴随着由算法保证的 **代价对等** 机制。
- **寂照堂**：每段完结人生凝结为一件「意象之物」，构成你个人的回响圣殿。

### 🕸️ 织渊（因果纠缠网络）
- **匿名因果纠缠**：当两个镜中人在同一时空相遇，且至少一方携带干预印记，系统即生成一条不可见的因果丝线。
- **共鸣域**：大量纠缠聚集后自动生成的共享平行世界，多个用户的镜中人共同生活、相互影响。
- **众生低语**：在他人面临人生重大抉择时，你可以匿名投送一枚人生回响，微弱地影响对方世界的风向。
- **神话域**：由系统从集体行为中自动识别并书写的「世界线传奇」，永久收藏于 **无尽图书馆**。

### 🧠 技术亮点
- **多层级 AI Agent 架构**：系统主控 Agent、世界 Agent、镜中人 Agent、叙事 Agent 协同工作。
- **因果图谱数据库**：所有事件与干预均存储为因果链，确保世界线自洽。
- **事件驱动的模拟时钟**：支持 1:1 实时、快进、跳转关键事件三种时间模式。
- **隐私优先的匿名化管道**：所有跨用户数据均不可逆匿名化，用户拥有完整的「被遗忘权」。

## 🚀 快速开始（开发者预览）

> ⚠️ 项目当前处于 **Pre-Alpha** 阶段，核心架构仍在快速迭代。以下步骤仅供早期贡献者搭建开发环境。

### 环境要求
- Python 3.10+
- Node.js 18+
- Docker (推荐)
- Neo4j 5.x

### 本地启动

```bash
# 克隆仓库
git clone https://github.com/your-org/mirror-abyss.git
cd mirror-abyss

# 启动基础设施（数据库、队列）
docker-compose up -d

# 安装后端依赖
cd backend
pip install -r requirements.txt

# 初始化数据库
python manage.py init-db

# 启动后端服务
uvicorn main:app --reload

# 新开终端，启动前端
cd ../frontend
npm install
npm run dev
